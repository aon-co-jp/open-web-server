open-web-server

Apache+Nginxハイブリッド仕様のRust製Webサーバー。マルチテナントHostルーティング・ TLS終端(SNIごとの証明書切替+ACME自動取得)・静的ファイル/PHP配信・固定IP不要のDDNS更新に対応。

ダウンロード・インストール

🐧 Linux

AlmaLinux・Ubuntu・Debian・Fedora・RHEL等、systemdを使う主要ディストリ共通

curl -fsSL https://github.com/aon-co-jp/open-web-server/releases/latest/download/open-web-server-linux-x86_64.tar.gz | tar xz
sudo ./install.sh
sudo systemctl edit open-web-server
sudo systemctl enable --now open-web-server

🪟 Windows / Windows Server

管理者権限のPowerShellで実行

Invoke-WebRequest -Uri "https://github.com/aon-co-jp/open-web-server/releases/latest/download/open-web-server-windows-x86_64.zip" -OutFile ows.zip
Expand-Archive ows.zip -DestinationPath open-web-server
cd open-web-server
.\install.ps1

📱 Android(スマホ・タブレット)

ARM64(arm64-v8a)/x86_64両ABI対応。起動時に省メモリ版・省電力版・通常版・常時電源接続版の 4電源プロファイルから選択でき、選択に応じてホーム画面アイコンの表示も切り替わります。

open-web-server-android-debug.apk をダウンロード

注意: 現時点ではデバッグ署名のAPKです(正式な配布用署名鍵は未整備)。インストール時は「提供元不明のアプリ」の許可が必要です。実運用では正式な署名鍵での再ビルドを推奨します。
